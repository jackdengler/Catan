# Team logos

Drop a square PNG (transparent background works best) for each team here, named
exactly:

- steelers.png
- ravens.png
- orioles.png
- yankees.png

They're rendered as round crests anywhere a themed player shows up (scoreboard,
hand, lobby roster, mini-scores). If a file is missing, that team falls back to
its monogram crest (PIT / RAV / ORI / NYY) automatically — so you can add them
one at a time.

SVGs also work; change the extension in client/src/game/TeamBadge.tsx (the LOGO
map) if you prefer .svg.
